# Zomato Dataset Analysis

An exploratory data analysis project completed as part of my Data Science Internship at InternSpark.

## 📌 Project Overview
This project analyzes a dataset of over 56,000 restaurants from Zomato to extract insights regarding location hotspots, price-vs-rating trends, and popular cuisines.

## 🛠️ Tech Stack Used
* **Environment:** Google Colab 
* **Language:** Python 3.x
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, WordCloud

## 📊 Core Tasks Completed
1. **Data Cleaning:** Handled messy text fields, removed irregular strings from rating columns, and managed missing values.
2. **Exploratory Data Analysis:** Analyzed key relationships such as Cuisine vs Rating and Price vs Rating.
3. **Data Visualization:** Built distribution bar charts, scatter plots, correlation heatmaps, and cuisine word clouds.
4. **Business Strategy:** Formulated 5 data-driven market recommendations for food-tech platforms.
