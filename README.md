# Multi-Phase Data Science & Analytics Internship (InternSpark)
Welcome to my portfolio repository for the InternSpark internship program. This repository documents an end-to-end engineering journey spanning data cleansing pipelines, exploratory analysis, and predictive machine learning models.

# TASK 1 : Zomato Dataset Analysis
## 📌 Project Overview
This project analyzes a dataset of over 56,000 restaurants from Zomato to extract insights regarding location hotspots, price-vs-rating trends, and popular cuisines.

## 🛠️ Tech Stack Used
* **Environment:** Google Colab 
* **Language:** Python 3.x
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, WordCloud

## 📊 Core Tasks Completed
1. **Data Cleaning:** Handled messy text fields, removed irregular strings from rating columns, and managed missing values.
2. **Exploratory Data Analysis:** Analyzed key relationships such as Cuisine vs Rating and Price vs Rating.
3. **Data Visualization:** Built distribution bar charts, scatter plots, correlation heatmaps, and cuisine word clouds.
4. **Business Strategy:** Formulated 5 data-driven market recommendations for food-tech platforms.




# 🤖 Task 2: Loan Approval Prediction System
Objective: Designed an automated credit-risk evaluation framework using ensemble machine learning classifiers to predict application outcomes with high precision.

## Tech Stack Used 
* **environment:** Google Colab
* **Language:** Python 3.x
* **Libraries** Pandas, NumPy, Matplotlib, seaborn, scikit learn.

  
### Key Milestones
* Built an automated preprocessing pipeline utilizing `LabelEncoder` for categorical strings and `StandardScaler` to bring distinct numeric ranges onto a uniform playing field.
* Solved target label class imbalances natively inside the models by applying weighted distributions (`class_weight='balanced'`).
* Trained and compared a baseline Logistic Regression model against an advanced **Random Forest Classifier**, achieving a dominant **94.81% evaluation accuracy** score.
